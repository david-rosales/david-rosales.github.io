//Projects

The projects I am currently working on or have finished are listed below with short descriptions about them. Along with the description is a button titled VIEW. Clicking the button will take you to a page with more information on the project. Not all projects I have made are listed.

Save The Humans (STH)

STH is a game being created in Java with the Swing library. There has been plenty of work on the game so far. With a currently working arcade mode, the game is a lot of fun!

World War II: Artillery

This game was created for the 2015 MESA Team Greenfoot Competition. Coded using Java and Greenfoot, the player controls a Soviet tank in algebraic warfare.

Minecraft Plugin: Trading

A Bukkit server modification for Minecraft that enables fair and easy trading. Using an inventory made GUI, players can trade through an interface.

Minecraft Plugin: Slender

A Bukkit server modification for Minecraft that recreates the environment from the hit horror game Slender. Using this plugin, server managers can create multiple arenas for players to use.

Minecraft Plugin: EasyPunishments

A Bukkit server modification for Minecraft that allows server administrators to manage player punishments through an inventory made GUI.

Minecraft Minigame: Castle Siege

A Bukkit server minigame for Minecraft created for Mineplex when it first began. Players consistently seiged enemy castles for some time to win.
